Assignment 1 - CGPA Calculator

Objective : To practice web page design & client side scripting using only HTML, CSS and JavaScript

Assoc. Prof. Dr. Tamer TULGAR

In this assignment the duty to fulfill is to design a web page which will accept course info from the user and show all of the entered courses in a table. At the end of the table the current cgpa calculation should be shown.
The course entry, dynamic table generation and the cgpa calculation should be triggered by a button click.
Students should pay attention and experience that the HTTP is a stateless protocol. Hence, navigation away from the page should be blocked.
Also, the entered data will be continuously used in a calculation. Therefore, the data should not only be displayd as a table row but also should be stored in a array.
